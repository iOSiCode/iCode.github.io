Course Outline

   WEEK I:
     Sunday: Create APP I
       – Programming fundamentals review and Swift language syntax.
       – xCode tour and creating the first User Interface (UI).
       – Connecting UI elements to code.
       – Setting-up GitHub.

     Monday: Create APP II
       – Auto-layout and constraints.
       – Pop-up alerts.
       – Challenge I.

     Tuesday: Start APP III
       – Create your own assets through figma.
       – Using guard-statement for validation.
       – Navigating between multiple screens using .show() method.
       – Using UICollectionView.
       – Introduction to Protocols and Delegates.
       – Custom CollectionView Cell.

     Wednesday: Continue APP III and start APP IV
       – Adding animation.
       – Using timers.
       – Challenge II.

     Thursday: Continue APP IV
       – Challenge III.


   WEEK II:
     Sunday: Create APP V
       – Explaining API calls.
       – What is a JSON?
       – Parsing JSON's.
       – Navigating between screens using NavigationController
       – Custom Protocols.

     Monday: Start APP VI
       – Calling API's using a unique key.
       – Using UITableView.
       – Using Concurrence with DispatchQueue.

     Tuesday: Continue APP VI
       – Increase app efficiency by storing reusable data.
       – Adding App icon.

     Wednesday: Create APP VII
       – Pair project.
       – App walkthrough presentation.

     Thursday: Create APP VIII
       – Individual project.
       – App walkthrough presentation.
       – What is next in your journey?

Pre-Requisites

   - A Mac product (MacBook, Mac or iMac)
   - xCode – Free from the app-store
   - Discord – Free software (Download Discord)
       Note: Discord is a free software, mainly used for creating comunities. It allows
        multiple user to communicate through messages, voice calls or video calls.
   - Basic understanding of programming (Variables, if-statements, loops, functions)
        Note: You do not need to implement the concepts, just be familiar with them.
